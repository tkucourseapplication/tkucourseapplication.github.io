Windows 桌面版選課程式

在使用此程式前，請先注意以下事項:

  1. 學校系統的一次只接受一個地方登入選課限制，請勿在使用程式期間該帳號登入選課以避免不必要的程式錯誤!!
  2. 此程式需要安裝 Microsoft .NET Framework 4.5 以上(含)才能夠正常執行，若你尚未安裝請點這裡來下載進行安裝
  3. 因發佈此程式前有對執行檔進行加密，對於某些較敏感防毒軟體會掃出有病毒是屬於正常現象，請放心使用。

使用說明

介面內容說明 :

  1. 學號 : 輸入要選課的學號
  2. 密碼 : 輸入的"選課專用"密碼
  3. 模式 : 選課模式分兩種，一般加選(搶課適用)，迴圈加選(沒選到其他時候適用)
  4. 伺服器 : 選擇要在中文或是英文版網頁選課

而本程式是採用部分符號表式來讓程式知道你的選課流程，選課的方式有分2種，輸入的方式說明如下 :

  1. 一般加選(Normal) :


    2. **此模式為搶課用途於選課開放前幾分鐘按下開始執行鍵即可(開放選課時間前一分鐘按最佳)!!

    此模式之間的分隔符號判斷為“>”, “#” ,”,” 和換行來表示
    “>” 表示你的選課排順位區隔
    “#” 表示要退選此課程，加在4位數代碼前面，請小心使用!!
    ”,” 和 換行表示區間分開

    輸入範例如下:

    0000>1111>2222,#3333,4444,5555>6666,7777>#8888>9999,1234>#2345>3456
    或是

    0000>1111>2222

    以上2種輸入方式所代表的意思相同，請擇一方法使用
    假設 0000,4444,5555,6666,7777 選課人數已滿，電腦執行的方式如下 :

    • 加選 0000 (滿，使用備案)
    • 加選 1111(成功, 之後備案捨棄直接跳下一區間)
    • 退選 3333
    • 加選 4444(滿, 無備案直接跳下一區間)
    • 加選 5555 (滿，使用備案)
    • 加選 6666(滿, 無備案直接跳下一區間)
    • 加選 7777(滿，使用備案)
    • 退選 8888(退選完後看下一個備案)
    • 加選 9999(成功, 之後備案捨棄直接跳下一區間)
    • 加選 1234(成功, 之後備案捨棄直接跳下一區間)
    • 結束

  2. 迴圈加選(Loop) :

    ”,” 和換行表示區隔每個科目代碼
